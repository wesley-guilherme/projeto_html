Tags p, mark e small


A Tag P serve para colocar PARÁGRAFO em um texto. Exemplo:
Eu estou digitando essse texto, porém quero um parágrafo assim que eu terminar esta linha;

Agora pronto já comecei uma outra linha.

A tag MARK é um MARCA TEXTO indicando algo importante na leitura. Exemplo:
A linguagem de programação mais utilizada em 2023 é PHYTON seguida de JAVASCRIPT

A tag SMALL é para deixar a font menor de um intervalo de texto definido. Exemplo:

Tags i, u, a e strong


A tag i pode representar uma parte do texto com uma diferença semântica, na qual a representação tipográfica padrão é no tipo itálico.
Exemplo: O favorito a Copa de 2026 é a ARGENTINA

A tag u representa a parte sublinhada em um texto
Exemplo: O favorito a Copa de 2026 é a ARGENTINA

A tag a serve para cria um link com outro documento podendo apontar qualquer recurso na web desde uma página html a um filme.
Exemplo: Página do Google

A tag strong dá ao texto uma forte importância, e é tipicamente mostrado em negrito.
Exemplo: O favorito a Copa de 2026 é a ARGENTINA

Tags ol, ul e li


Lista é um importante recurso de HTML, pois permite criarmos tópicos de textos para uma melhor exemplificação de um determinado assunto. São recursos extremamente usados, inclusive quando nem imaginamos que ele está sendo usado, como no caso de menus. Hoje, boa parte dos menus em HTML é feito com listas.

ol: representa uma lista de itens ordenados:
  1. Siafw
  2. SiafController
  3. SiafGourmet

ul: representa uma lista de itens não ordenados:
  • Siafw
  • SiafController
  • SiafGourmet

Tags hr, sub, sup e blockquote


A tag hr epresenta uma quebra temática entre elementos de nível de parágrafo representa tado por uma linha horizontal (por exemplo , uma mudança da cena de uma história, ou uma mudança de tema com uma seção).

Exemplo: Estamos vivenciando 2 Guerras entre
Russia e Ucrania
Israel e Faixa de Gaza

A tag sub especifica o texto embutido que deve ser exibido como subscrito apenas por motivos tipográficos. Os subscritos normalmente são renderizados com uma linha de base reduzida usando texto menor

Exemplo: Estamos falando da molécula H2O

A tag sup especifica o texto embutido que deve ser exibido como sobrescrito apenas por motivos tipográficos. Os sobrescritos normalmente são renderizados com uma linha de base elevada usando texto menor

Exemplo: 32=9 --- 52=25

A tag blockquote indica que o texto incluído é uma longa citação. Normalmente, este é processado visualmente pelo recuo da do texto.

Exemplo:

Esta é uma citação tirada do site do Clube de Regatas Flamengo
